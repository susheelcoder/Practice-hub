HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of web pages using markup.

Basic HTML Document 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

Document Structure

<!DOCTYPE html>

Defines the document type and HTML version

<html>

Root element of an HTML page

<head>

Contains meta information about the document

<body>

Contains the visible page content

Semantic HTML

Note: Semantic HTML improves accessibility and SEO by clearly describing the meaning of content.
Semantic Layout

    <h1>Website Title</h1>
    <nav>
        <a href="#">Home</a>
        <a href="#">About</a>
    </nav>
</header>

    <article>
        <section>
            <h2>Article Title</h2>
            <p>Article content...</p>
        </section>
    </article>
    
    <aside>
        <h3>Related Links</h3>
    </aside>
</main>

<footer>
    <p>© 2024 My Website</p>
</footer>

HTML Best Practices

1. Use Semantic HTML

Always use semantic elements like <header>, <nav>, <main>, <article>, <section>, <aside>, and <footer> instead of generic <div> elements.

2. Always Include Alt Text for Images

Every <img> tag should have an alt attribute for accessibility and SEO.

3. Use Proper Document Structure

Always include <!DOCTYPE html>, proper meta tags, and follow heading hierarchy (h1 to h6).
